# Soressa

The "Bus Tracking device"

This module runs on ESP8266 and send messagens to a MQTT Broker.

## Requirements

- ESP8266 (ESP-12 or ESP-7)
- NodeMCU Firmware
- GPS Module
- RTC Module

## NodeMCU required modules

- bit
- file
- GPIO
- i2c
- MQTT (with SSL)
- node
- timer
- UART
- WiFi
